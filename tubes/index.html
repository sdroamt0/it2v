Il Primo

DIY - Vacuum Tube Amplifier by Giuseppe Amato


Technical Specifications

  • Push-Pull stereo integrated amplifier
  • Max Output Power 2x ~30W
    • @ 1 Volt peak input signal, with 6dB global negative feedback
    • @ 0.5 Volt peak input signal, with no negative feedback
  • Power Stage: 2x 2 EL34/6AC7 power vacuum tubes in Push-Pull Ultra-Linear configuration and fixed bias
  • Phase Split Stage: DC coupled Concertina with 12AX7 vacuum tubes
  • Input Stage: 12AX7 vacuum tubes with self bias and bypass capacitor
  • Output Transformer: Custom
    • Primary: 7600 Ohm C.T., 40% screen tap
    • Secondary: 4, 8, 16 Ohm
  • Power Supply Unit Transformer: Custom 650V C.T. @400mA
  • Global Negative Feedback: max 6dB adjustable
  • Two only capacitors in the signal path
  • 4, 8, 16 Ohm Speakers
  • Input Impedance: 100K Ohm
  • 6 selectable input sources

Configuration

  • Power Stage: Electro Harmonix 6AC7
  • Phase Split Stage: Sovtek 12AX7-LPS
  • Input Stage: Sovtek 12AX7-LPS
  • Class AB with 40 mA anode current bias
